Copy this into a NEW file named ".env" in this same folder, then fill in values.
(The bot and podcast_to_notion.py both read ".env" automatically.)

----------------------------------------------------------------------
# From @BotFather ("Use this token to access the HTTP API: ...")
TELEGRAM_BOT_TOKEN=123456789:AA...your-bot-token...

# Your Telegram chat ID(s), comma-separated. Locks the bot to just you.
# Don't know it yet? Leave blank, start the bot, message it once -> it replies with your ID.
TELEGRAM_ALLOWED_CHAT_IDS=

# Notion internal integration token (notion.so/my-integrations).
# Share the "Podcast Notes" database with this integration.
NOTION_TOKEN=ntn_...your-notion-token...

# Optional overrides:
# NOTION_DB_ID=<YOUR_NOTION_DATABASE_ID>
# NLM_BIN=<HOME_PATH>\.local\bin\nlm
----------------------------------------------------------------------
